# Skill: /classify — 品类自动识别

## 功能

根据器件名称、材料、工艺，自动识别器件所属品类，输出品类 + 置信度 + 推理依据。

## 输入

```
器件名称: <必填>
材料: <选填>
工艺: <选填>
```

## 执行流程

### Step 0: 复合体识别

在正式匹配前，先检查器件是否为多组件复合体：
- 名称含"三合一"/"组件"/"模组"/"+"（如"电池盖+TP+miniLED"）
- 材料跨两个以上品类（如同时含PC+GF结构材料+PET光学材料+FPC安规材料）
- 工艺含"贴合"/"组装"/"复合"且涉及不同品类组件

复合体识别输出：标注"多品类复合"，在后续流程中对每个子组件分别进行品类识别。

### Step 1: 名称关键词匹配

检查器件名称是否命中以下直接匹配表：

| 关键词（器件名称含） | 品类 |
|---------------------|------|
| 中框 / 面壳 / 电池盖 / 侧键 / 支架 / 螺丝 / 卡托 / 按键 / 镜头支架 / 炮筒 | 结构件 |
| 双面胶 / 导电布 / 导电泡棉 / 导热垫 / 导热硅胶 / 绝缘片 / 防水透气膜 / 标签 / 保护膜 / 铜箔 / 石墨片 / NFC铁氧体 / 泡棉 / 麦拉 | 辅料 |
| 扬声器 / 受话器 / 听筒 / 麦克风 / 马达 / 蜂鸣器 / 线性马达 / 音腔 | 电声器件 |
| 摄像头 / 闪光灯 / 红外传感器 / 距离传感器 / 光感 / TOF / 镜片 / 镜头 | 光学器件 |
| 彩盒 / 吸塑 / PE袋 / 说明书 / 标签纸 / 纸托 / 缓冲材 / EPE / 气泡袋 / 卡通箱 | 包材 |
| 电池 / 充电器 / 数据线 / 保险丝 / FPC / FFC / 天线 / 电源适配器 / 无线充电线圈 | 安规器件 |

### Step 2: 材料关键词辅助匹配

如果名称未命中或存在歧义，用材料做辅助判断：

| 关键词（材料含） | 倾向品类 |
|-----------------|---------|
| PC+GF / PC+ABS / 铝合金 / 镁合金 / 不锈钢 / 钛合金 / 陶瓷 / 玻璃 / 复合板 / PMMA+PC | 结构件 |
| 双面胶 / 泡棉 / 导热硅胶 / 导电布 / PET / PI膜 / 铜箔 / 铝箔 / 石墨片 | 辅料 |
| PU / PET / PEN（振膜） / 磁钢 / 音圈 / 钕铁硼 | 电声器件 |
| 蓝宝石 / 光学玻璃 / PC（光学级） / PMMA / IR滤光片 | 光学器件 |
| 瓦楞纸 / 白卡纸 / PET（吸塑） / EPE / EVA / 珍珠棉 | 包材 |
| 锂离子 / Li-Po / 锂聚合物 / FPC基材 / LCP | 安规器件 |

### Step 3: 工艺关键词辅助匹配

| 关键词（工艺含） | 倾向品类 |
|-----------------|---------|
| 注塑 / CNC / 压铸 / 冲压 / 阳极氧化 / 喷涂 / 电镀 / PVD / 热压 / 贴合 | 结构件 |
| 模切 / 分条 / 覆合 / 涂布 / 贴合（辅料） | 辅料 |
| 充磁 / 点焊（音圈） / 绕线 / 热压成型（振膜） | 电声器件 |
| 镀膜 / 贴合（镜片） / 精密注塑（光学） / IR镀膜 | 光学器件 |
| 裱纸 / 模切 / 吸塑成型 / 印刷（包材） | 包材 |
| SMT / 回流焊 / 绕线（线圈） / 层压（FPC） | 安规器件 |

### Step 4: 综合打分 + 歧义处理

- 名称命中 + 材料命中 + 工艺命中 → confidence: **high**
- 名称命中 + 仅一项辅助命中 → confidence: **medium**
- 仅名称命中，材料和工艺不匹配 → confidence: **medium**, 标注 alternatives
- 名称未命中，仅靠材料/工艺推断 → confidence: **low**, 标注"品类由AI推断"
- 名称跨品类模糊 → 根据材料/工艺区分（例："支架"可以是结构件也可以是辅料支架）

## 输出格式

```json
{
  "category": "结构件",
  "confidence": "high",
  "reasoning": "器件名'电池盖'直接命中结构件关键词，材料复合板和工艺UV转印均为结构件典型特征",
  "alternatives": []
}
```

## 边界条件处理

| 场景 | 处理方式 |
|------|---------|
| 名称含"支架" | 材料含金属/塑胶→结构件；材料含导电布/泡棉→辅料 |
| 名称含"标签" | 纸质标签→包材；功能性标签（防水/导电）→辅料 |
| 名称含"镜头" | 光学镜头→光学器件；镜头装饰圈→结构件 |
| 名称含"线圈" | 无线充电→安规器件；扬声器音圈→电声器件 |
| 名称含"三合一"/"组件"/"模组"/"+" | 识别为多品类复合体，输出主品类+标注"多品类复合" |
| 名称+材料+工艺均为空 | 返回 category:"无法识别"，confidence:"fail"，reasoning:"需用户提供器件名称、材料或工艺信息" |
| 材料/工艺均为空 | 仅靠名称判断，标注 confidence: medium |
| 多品类同时命中 | 按得分排序，取最高分；备选列在 alternatives 中 |
